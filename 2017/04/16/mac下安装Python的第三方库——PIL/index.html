mac下安装Python的第三方库——PIL

首先自己在安装这款遇到了很多坑，不过最后还是搜索引擎救了我。感谢大佬！！

首先申明，系统针对于mac，其它系统请绕开！！！这也是我目前发现最为简介的安装方法，相比于其它的下载安装，出的错误最少。不过归根结底都是一个’X11/Xlib.h’ file not found的错误。

  • 首先你copy运行sudo easy_install -f http://www.pythonware.com/products/pil/ Imaging

  • 如果第一次安装，不出意外，都会出现如下❌

  • 此时执行xcode-select --install会跳出让你安装一个软件，安装成功的话就㊗恭喜你，离成功还有一步。

  • 执行sudo pip install -U Pillow安装Pillow即可。这里提出两点，因为电脑自带的是python 2.7，你又下载了3.6，此时处于混编状态，你可以执行sudo pip install -U Pillow安装。

  • 这时你可以写写来测试下看是否安装成功
